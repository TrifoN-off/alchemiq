# Security Policy

## Supported versions

alchemiq is in the 0.x series. Security fixes are released against the latest
0.x version only.

| Version | Supported |
|---|---|
| 0.1.x | yes |
| older | no |

## Reporting a vulnerability

Please do not open a public issue for security problems.

Use GitHub private vulnerability reporting: open the repository **Security** tab
and choose **Report a vulnerability**. If you prefer email, write to
trifonov.nikit@gmail.com.

Include as much detail as you can:

- a description of the issue and its impact,
- steps to reproduce or a proof of concept,
- affected versions and environment.

You can expect an acknowledgement within a few days. Once a fix is ready it
ships in a new 0.x release, with credit to the reporter unless you ask to stay
anonymous.
